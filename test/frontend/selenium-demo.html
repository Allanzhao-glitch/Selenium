🎯 元素定位练习

尝试使用不同的定位方式找到这些元素：

ID 定位

Class 定位

Name 定位

XPath 定位

data-testid 属性

定位示例代码：

# ID 定位
driver.find_element(By.ID, "username-input")
# Class 定位
driver.find_element(By.CLASS_NAME, "form-input")
# XPath 定位
driver.find_element(By.XPATH, "//input[@data-testid='user-profile']")
# CSS Selector 定位
driver.find_element(By.CSS_SELECTOR, "input[data-feature='newsletter']")
